A lightweight **Angular (Standalone Components)** application for managing staff and their leave requests.  
Includes separate workflows for **HOD** and **Staff**, using clean Bootstrap UI and LocalStorage-based data.

---

## Features

### HOD
- Add, view, and delete staff  
- Approve / reject leave requests  
- Search, sort, and basic pagination  
- Bootstrap modals for confirmations  

### Staff
- Apply for leave  
- Dynamic date validation  
- View all applied leaves  
- View detailed leave info in a modal  

---

## Tech Stack
- **Angular 17+ (Standalone Components)**
- **Bootstrap 5**
- **TypeScript**
- **LocalStorage (mock backend)**
